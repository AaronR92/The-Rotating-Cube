## 3D Rotating Cube
### Controls
WS keys for rotation by x-axis.   
AD keys for rotation by y-axis.   
QE keys for rotation by z-axis.   
R - autorotation
### Requirements
Java 1.8 or higher
### Compile
1. Navigate to project folder
2. `./gradlew clean`
3. `./gradlew jar`
Output will be stored in `build/libs`
### Run 
`java -jar file_name.jar`
